# Screenshot Images

Add the following screenshots to this folder:

| Filename | Description |
|----------|-------------|
| `banner.png` | Project banner/logo (recommended: 1200x400px) |
| `landing-page.png` | Landing/home page screenshot |
| `student-dashboard.png` | Student dashboard view |
| `instructor-dashboard.png` | Instructor dashboard view |
| `admin-panel.png` | Admin panel/dashboard |
| `course-management.png` | Course creation/management page |
| `quiz-interface.png` | Quiz taking interface |

## How to Capture Screenshots

1. Run the application locally
2. Navigate to each page
3. Use browser developer tools (F12) to set viewport to 1280x800
4. Take screenshots using:
   - **Windows:** `Win + Shift + S`
   - **Browser:** Right-click > Capture screenshot (Firefox)
   - **Chrome DevTools:** `Ctrl + Shift + P` > "Capture screenshot"

## Recommended Dimensions

- **Banner:** 1200 x 400 pixels
- **Screenshots:** 1280 x 800 pixels (or your actual viewport)
- **Format:** PNG for quality, or WebP for smaller size
