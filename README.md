## Replacing with Real Screenshots

To replace these mockups with actual screenshots:

1. Run the application locally
2. Navigate to each page
3. Capture screenshots at 1280x800 resolution
4. Save as PNG and replace the corresponding SVG files
5. Update README.md to reference `.png` instead of `.svg`

## Screenshot Tips

- **Windows:** `Win + Shift + S`
- **Browser DevTools:** `Ctrl + Shift + P` > "Capture screenshot"
- **Firefox:** Right-click > Take Screenshot
